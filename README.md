# 📱 محرر التطبيقات - App Editor

## مرحباً بك في محرر التطبيقات الاحترافي! 🎉

تطبيق ويب حديث لتصميم واجهات محررات التطبيقات بنمط **LowCode** مشابه لـ **Lovable** و **FlutterFlow**.

---

## ✨ المميزات الأساسية

### 🎨 **التصميم الاحترافي**
- تصميم أسود فاخر مع تدرجات ممتعة
- واجهة مقسومة لنصفين (Split View)
- تأثيرات انتقالية سلسة
- دعم كامل للعربية (RTL)

### 📝 **محرر النص المتقدم**
- حقل إدخال كبير وواضح لكتابة وصف التطبيق
- عداد أحرف فوري (حد أقصى 1000 حرف)
- حفظ تلقائي في LocalStorage
- اختصارات لوحة مفاتيح ذكية

### 👁️ **معاينة فورية**
- تحديث فوري أثناء الكتابة
- استخراج ذكي للميزات من الوصف
- اقتراح التقنيات تلقائياً
- تقدير مستوى التعقيد

### 💾 **إمكانيات الحفظ والتصدير**
- حفظ محلي في المتصفح
- تحميل المشروع كملف Markdown
- استرجاع المشاريع السابقة

---

## 🚀 كيفية الاستخدام

### 1. **افتح الملف**
```bash
# طريقة مباشرة
file:///path/to/index.html

# أو استخدم خادم محلي (مفضل)
python -m http.server 8000
# ثم افتح: http://localhost:8000
```

### 2. **ابدأ بكتابة الوصف**

في الجهة **اليسرى**، اكتب وصف تطبيقك بالتفصيل:

```
أنشئ لي تطبيق قائمة مهام متقدم يحتوي على:
- حقل إدخال لإضافة مهام جديدة
- قائمة تعرض جميع المهام مع حالتها
- إمكانية وضع علامة على المهام المكتملة
- زر لحذف المهام
- فلترة المهام حسب الحالة (مكتملة/قيد الانجاز)
- عدّاد للمهام المكتملة والمتبقية
- حفظ البيانات محلياً
```

### 3. **شاهد المعاينة**

في الجهة **اليمنى**، ستظهر فوراً:
- ✓ اسم التطبيق المقترح
- ✓ الميزات المستخرجة
- ✓ التقنيات المقترحة (HTML, CSS, JavaScript, React, إلخ)
- ✓ مستوى التعقيد (مبتدئ/متوسط/متقدم)
- ✓ اقتراحات للتحسين

### 4. **احفظ المشروع**

- **💾 حفظ**: ينزل ملف Markdown بمواصفات التطبيق الكاملة
- **🔄 تحديث**: يحدّث المعاينة يدوياً
- **إعادة تعيين**: يمسح جميع البيانات

---

## ⌨️ اختصارات لوحة المفاتيح

| الاختصار | الفعل |
|---------|-------|
| `Ctrl/Cmd + S` | حفظ الملف |
| `Ctrl/Cmd + R` | إعادة تعيين |
| `Ctrl/Cmd + E` | تحديث المعاينة |

## 🛠️ التقنيات المستخدمة

- **JavaScript ES6+**: منطق ذكي لاستخراج البيانات وتحديث المعاينة
- **LocalStorage**: حفظ تلقائي للمشاريع

## 📊 مثال على الاستخراج الذكي

**الإدخال:**
```
أنشئ تطبيق متجر إلكتروني بسيط باستخدام React و Firebase
- عرض المنتجات في شبكة
- سلة التسوق
- نظام الدفع
- قاعدة بيانات للمنتجات
- مصادقة المستخدمين
```

**المخرجات (المعاينة):**
- **الاسم**: متجر إلكتروني بسيط
- **الميزات**: [عرض المنتجات، سلة التسوق، نظام الدفع، إلخ]
- **التقنيات**: React, Firebase, JavaScript
- **المستوى**: متقدم 🔴
- **الاقتراحات**: [أضف اختبارات، وثق الكود، إلخ]

## 🎨 التخصيص

### تغيير الألوان

في `styles.css`، عدّل متغيرات الألوان:

```css
:root {
    --bg-primary: #0f0f0f;      /* اللون الأساسي */
    --accent: #6366f1;           /* لون التركيز */
    --accent-hover: #4f46e5;     /* عند التحويم */
    /* ... إلخ */
}
```

### تغيير الخطوط

في `styles.css`:

```css
body {
    font-family: 'Your Font Name', sans-serif;
}

.prompt-textarea {
    font-family: 'Your Code Font', monospace;
}
```

## 🐛 استكشاف الأخطاء

### المعاينة لا تتحديث
- ✓ تأكد من كتابة النص في حقل الإدخال
- ✓ اضغط زر "تحديث"
- ✓ افتح Developer Tools (F12) وتحقق من الأخطاء

### الحفظ لا يعمل
- ✓ تأكد من أن LocalStorage مفعّل في متصفحك
- ✓ جرّب متصفح مختلف
- ✓ امسح ذاكرة ��لتخزين المؤقتة

### المظهر غير صحيح
- ✓ حدّث الصفحة (Ctrl/Cmd + F5)
- ✓ امسح ذاكرة التخزين المؤقتة (Ctrl/Cmd + Shift + Delete)
- ✓ استخدم متصفح حديث (Chrome, Firefox, Safari, Edge)
